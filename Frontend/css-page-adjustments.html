Problem 2: Visibility and Display Properties

Visibility Properties

visibility: visible - This element is visible (default behavior)
visibility: hidden - This element is hidden but still takes up space
visibility: collapse - This element is collapsed (mainly for table elements)

Display Properties

display: none - The following element is completely removed from layout: This text is hidden with display: none (You can't see it!)
display: inline - This text is inline and flows with other text.
display: block - This element takes up the full width available.
display: contents - This container's box is not rendered, but children are. Child 1 Child 2
display: list-item - This element appears as a list item with bullet.

Problem 4: Padding and Margin Differences

Margin Example

Margin: 30px 20px 15px 40px
Margin creates space OUTSIDE the element's border. This element has different margins on each side:
  • Top: 30px
  • Right: 20px
  • Bottom: 15px
  • Left: 40px

Padding Example

Padding: 25px 15px 35px 45px
Padding creates space INSIDE the element's border. This element has different padding on each side:
  • Top: 25px
  • Right: 15px
  • Bottom: 35px
  • Left: 45px
